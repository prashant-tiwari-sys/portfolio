# Prashant Tiwari - Personal Portfolio

A clean, modern, and responsive personal portfolio website built with pure **HTML, CSS, and JavaScript**. Features smooth animations, scroll effects, and a professional light theme.

### Live Demo
[View Live Site](https://prashant-tiwari-sys.github.io/[portfolio]/)  

### Features
- Fully responsive design (desktop, tablet, mobile)
- Fixed navigation with smooth scrolling
- Animated hero section with profile photo
- About section with personal details and career objective
- Skills displayed in organized cards
- Projects section with screenshots, descriptions, live demos, and GitHub links
- Subtle hover effects and scroll reveal animations
- No external frameworks — vanilla code only

### Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, CSS Variables, Animations)
- JavaScript (Smooth Scroll, Intersection Observer)
- Font Awesome for icons

### Showcased Projects
1. **Simple Calculator**  
   A responsive web calculator for basic arithmetic operations.

2. **BMI Calculator**  
   Instant BMI calculation with a clean and user-friendly interface.
